## 🚀 HOW TO RUN

### Quick Start (3 steps):

**1. Install Backend**
```bash
cd backend
npm install
```

**2. Install Frontend**
```bash
cd frontend
npm install
```

**3. Run Both Servers**
```bash
# Terminal 1 - Backend
cd backend && npm run dev
# Runs on http://localhost:5000

# Terminal 2 - Frontend
cd frontend && npm run dev
# Runs on http://localhost:5173
```

**That's it!** Open browser to http://localhost:5173
